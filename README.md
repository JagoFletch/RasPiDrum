# Pre-requisites
Raspberry Pi 4
Behringer U-PHORIA UMC22
DSI touchscreen
Raspberry Pi OS Bookworm 64 bit (full)

#### On a fresh installation of Raspberry Pi Os
``` nano ~/drumbrain-setup.sh ```
CTRL + X to save, then Y to confirm.

Give permissions with
``` chmod +x ~/drumbrain-setup.sh ```

#### Run it as Pi
``` bash ~/drumbrain-setup.sh ```
CTRL + X to save, then Y to confirm.
